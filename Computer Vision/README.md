# Computer-vision-project
Java eclipse Computer vision project

Need openCV 4.1.1

Mean filter, Median filter, Gaussian filter, LoG filter, High frequency emphasis filter

openCV 이용한 이미지 필터링(노이즈 제거, edge smoothing or emphasis)입니다.
